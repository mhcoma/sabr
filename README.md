# Sabr
Sabr is simple stack-oriented programming language.

# Build from source
```
git clone --depth=1 https://github.com/mhcoma/sabr.git
cd sabr
mkdir build
cd build
cmake ..
make
```

# How to run

## Compile to bytecode
```
sabrc {source file name} {output file name}
```

## Interprete bytecode
```
sabre {bytecode file name}
```
